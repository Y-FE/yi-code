# 前端代码生成


[![npm](https://img.shields.io/npm/v/yi-code.svg)](https://www.npmjs.com/package/yi-code) [![Build Status](https://travis-ci.org/Y-FE/yi-code.svg?branch=master)](https://travis-ci.org/Y-FE/yi-code)

## 安装
```
npm i yi-code -g
```

## 生成组件
```
# 从普通文本文件生成
yhtml gen -h

# 从 YAML 格式文件生成包含几个组件的组件
yhtml yml -h
```

## 查看组件对应名称
```
yhtml -s
```
